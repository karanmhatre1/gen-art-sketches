"Character Stretch"

Monday, July 19, 2021
paper.js, dat.gui

Exploration of the letter 'A' through manipulation of its height, width, and crossbar position.

Press [1-8] keys to explore variations. Alteratively, use the controls on the top-right of the screen.

The letter is draw line by line using vectors instead of using a font so that we can manipulate each line independently.

Made a version that works with the LeapMotion. And hooked up some synth sounds for added effects.
